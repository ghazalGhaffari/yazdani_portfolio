Ajax Foundation's website ui design

AJAX Foundation

I had the privilege of designing the user interface for AJAX FOUNDATION, a groundbreaking platform that simplifies blockchain token issuance. AJAX's mission is to make blockchain technology accessible to everyone, enabling projects to fundraise by creating and selling their own tokens.

I crafted intuitive and visually appealing user interfaces for both web and mobile, enhancing the user experience. Additionally, I animated icons for this project, which you can see in action in the live version with the button below. Please note that the live version may differ in color as the project owner has recently transitioned to a dark theme, but the engaging and dynamic elements of the user interface remain unchanged.
